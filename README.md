# پروژه Frontend

این پروژه Frontend شرکت است که با React و Vite نوشته شده.

## چه تکنولوژی‌هایی استفاده شده؟

- **React** - کتابخانه JavaScript برای ساخت رابط کاربری
- **Vite** - ابزار مدرن برای توسعه و بیلد گرفتن (جایگزین سریع‌تر Create React App)
- **JavaScript** - زبان برنامه‌نویسی اصلی

## چطور پروژه رو روی سیستمم اجرا کنم؟

### ۱. نیاز دارید:

- Node.js (نسخه ۱۶ به بالا)
- npm (معمولاً با Node.js نصب میشه)

### ۳. کتابخانه‌ها رو نصب کنید:

```bash
npm install
or
yarn install
or
pnpm install
```

این دستور تمام کتابخانه‌هایی که پروژه بهشون نیاز داره رو نصب میکنه.

### ۴. پروژه رو اجرا کنید:

```bash
npm run dev
```

بعد از این دستور، برید تو مرورگر و آدرس `http://localhost:5173` رو باز کنید.

## دستورهای مفید

| دستور             | کارش چیه؟                        |
| ----------------- | -------------------------------- |
| `npm run dev`     | پروژه رو اجرا میکنه برای توسعه   |
| `npm run build`   | پروژه رو آماده production میکنه  |
| `npm run preview` | بعد از build، نتیجه رو نشون میده |

## چی هست این Vite؟

Vite یک ابزار مدرن هست که:

- خیلی سریع‌تر از webpack کار میکنه
- تغییرات کد رو فوری تو مرورگر نشون میده (Hot Reload)
- build گرفتن باهاش خیلی سریع‌تره
- جایگزین بهتری برای Create React App حساب میشه

## چطور تغییرات رو ببینم؟

وقتی `npm run dev` رو اجرا کردید، هر تغییری که تو کد بدید فوری تو مرورگر اعمال میشه. نیازی نیست server رو restart کنید.

## مشکل خوردید؟

- مطمئن شید Node.js نصب دارید: `node --version`
- اگه npm کار نمیکنه، `npm install` رو دوباره اجرا کنید
- پورت ۵۱۷۳ اشغال باشه، Vite خودکار پورت دیگه‌ای انتخاب میکنه

---

**نکته برای Backend ها:** این یک Single Page Application (SPA) هست که معمولاً از API های شما استفاده میکنه.
